-----------------------------------------------------------------------------------------------------------------
Sobrecarga:
-----------------------------------------------------------------------------------------------------------------

"É a capacidade de definir métodos para diferentes contextos, mas preservando seu nome"

mantém o nome, mas muda a lista de parâmetros

-----------------------------------------------------------------------------------------------------------------
Criação:
-----------------------------------------------------------------------------------------------------------------

alterar a assinatura do método:
Ass = nome + parâmetros

converterParaInteiro(float f);
converterParaInteiro(double d);
converterParaInteiro(String s);

-----------------------------------------------------------------------------------------------------------------
Exemplos: 
-----------------------------------------------------------------------------------------------------------------

println, valueOf

-----------------------------------------------------------------------------------------------------------------
Exercício:
-----------------------------------------------------------------------------------------------------------------

Crie uma aplicação que calcula a área dos 3 quadriláteros notáveis: quadrado, retângulo e trapézio.
Use sobrecarga.
